9.4 조건문 다루기

9.4.1 if, else, else if 문

  • if 문

    if 문은 if 뒤에 오는 소갈호 안의 조건식이 참으로 평가되면 중갈호 안의 코드를 실행하는 조건문이다.

  • else 문

    else 문은 if 문의 조건식이 거짓일 때 실행되는 블록문을 정의한다. 단독으로 사용할 수 없으며, 하나의 if 문에는 하나의 else 문만 사용할 수 있다.

  • else if 문

    else if 문은 if 문에 조건을 추가하고 싶을 때 사용한다. 개수에는 제한이 없으며 단독으로 사용할 수 없어, 항상 if 문 다음에 사용하고, else 문 이전에 사용해야 한다.

  • 중첩하기

    분기처리는 어떤 조건식을 만족할 때 어떤 블록문을 실행할지 결정하는 것을 말한다. if 문을 작성하려는 코드의 분기처리에 따라 중첩해서 사용하며, 중첩 횟수에는 제한이 없다.

9.4.2 switch 문

switch 문은 switch 뒤에 오는 소괄호 안의 값과 일치하는 case 문이 있을 때 해당 코드를 실행하는 조건문이다. 일치 여부 확인은 일치 연산자를 이용한 비교 연산처럼 값과 자료형을 함께 비교한다.
하나 이상의 case 문과 default 문, break 문을 사용한다. 의도적으로 break 문을 생략해서, case 문을 연속 실행할 수 있다.

9.4.3 if 문과 조건식

if 문은 조건에 식을 사용할 수 있다. 논리 연산자나 비교 연산자를 식에 사용할 수 있으며, 여러 조건을 추가하기 위해 논리연산자로 식과 식을 연결해 작성할 수 있다.

9.4.4 if 문과 switch 문

if 문은 범위를 이용한 조건을 작성할때, switch 문은 값이 하나일 때 사용하는 것이 적합하다.

9.5 반복문 다루기

반복문은 지정한 조건이 참으로 평가되는 동안 지정한 블록문을 반복해서 실행하는 문법이다. 반복문에는 while 문, do...while 문, for 문이 있다.

9.5.1 while 문

while 문은 특정 조건을 만족하는 동안 블록문을 실행한다.

9.5.2 무한 반복문

모든 반복문은 무한 반복문을 주의해야 하며, 무한 반복문이란 반복문의 조건이 계속 참으로 평가되어 반복문이 끝나지 않고 무한히 실행되는 것을 말한다.
따라서 항상 반복문의 조건이 거짓이 되는 순간이 있는지 신경 써야 한다.

9.5.3 do...while 문

do...while 문은 특정 조건이 참으로 평가되는 동안 do 다음에 오는 블록문을 반복 실행한다. while 문과의 차이점은 블록문을 한 번 수행한 후에 조건식을 평가한다는 특징이 있다.

for 문

for 문은 횟수를 지정해 횟수가 끝날 때 까지 블록문을 반복 실행하는 반복문으로, 조건식과 증감식이라는 독특한 실행 구조로 되어 있다.
for 문은 초깃값 - 조건식 - 블록문 - 증감식 - 조건식 순서로 실행횐다. 이때 초깃값은 최초 1회만 실행횐다. if 문과 같이 중첩해서 사용할 수 있다.

for 문과 배열

배열과 같은 자료형을 반복 횟수 용도로 사용할 수 잇으며, for 문으로 배열 요소에 접근해 값을 출력할 수 있다.

9.5.6 for...in

for 문의 소괄호 안에 in 키워드를 두고 키워드의 오른쪽에는 탐색의 대상이 되는 배열 또는 객체 리터럴, 왼쪽에는 배열 또는 객체 리터럴을 탐색해서 키를 저장할 가변수를 놓는다.

  • 객체 리터럴을 반복할 경우

    for...in 문으로 객체 리터럴을 반복 탐색하면 탐색 결과로 가변수에 객체 리터럴의 키가 할당되어 객체 리터럴의 키와 값을 출력할 수 있다.

  • 배열을 반복할 경우

    인덱스 순서대로 배열에 반복 접근한 것처럼 보이지만, 문법적으로는 배열의 순서대로 접근하는 것을 보장하지 않는다.

9.5.7 break 문

break 문을 사용하면 어떤 반복문이라도 종료되며 종료 조건을 만족하지 않더라도 인위적으로 반복문을 종료할 때 사용한다. if 문과 함께 사용하는게 일반적이다.

9.5.8 continue 문

continue 문은 반복문을 건너뛰고 실행하라는 명령이다. 반복문의 블록문에서 continue 문을 만나면 해당 반복 실행한 건너뛰게 된다.

셀프체크1

                    
let num = 0;
for(let i = 1; i < 1000; i++){
    if(i % 2 == 0) num++;
}
console.log("짝수의 개수는" + num  +"개 입니다.");

셀프체크2

                    
for(let i = 1; i <= 9; i++){
    for(let j = 1; j <= 9; j++){
        console.log(`${i} x ${j} = ${i*j}`);
    }
}

셀프체크3

                    
let num = 0;
let num2 = 0;
for(let i = 100; i <= 999; i++){
    num2 = 0;
    num = i;
    while(num != 0){
        num2 = num2 + (num % 10)**3;
        num = Math.floor(num / 10);
    }
    if(i == num2) console.log(`${i}는 암스트롱 수 입니다.`);
}
