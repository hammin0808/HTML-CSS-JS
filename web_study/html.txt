요소에 너비를 주고, 좌우 마진 값을 auto 로 주면 중앙으로 배치된다. 대신, 요소가 블록 속성이어야 한다.
태그에는 선택자 가중치가 존재하여 자칫하면 css 스타일이 적용되지 못할 수 있다.
ul li 태그를 사용할 때 counter 함수를 활용하고, list-style 속성을 none 으로 설정하여 리스트 스타일을 다르게 설정 할 수 있다.
inline-block 은 생각보다 유용하며, text-align: center 로 원하는 요소를 중앙으로 배치할 수 있다. inline 요소와는 다르게 마진이나 패딩 값을 넣을 수 있다.
따라서, inline 요소인 a 태그나 다른 인라인 요소들의 높이나 마진 패딩 값을 부여 할 수 있게 한다.
input 과 button 의 display 속성은 inline-block 이다.

input 태그와 button 태그는 inline-block 속성이기 때문에 이를 컨테이너로 감싸 text-align:center 값을 주어 중앙으로 배치할 수 있다.
이는 별도로 좌우 마진 값을 설정할 필요가 없게 된다.

개인적으로 display:flex 속성을 주어 justify-content: center를 하는게 더 편리해 보인다. 이것은 별도로 좌우 마진 값을 지정하지 않아도 중앙으로 배치해준다.

float 속성은 부모가 자식요소의 범위를 보장해주지 않기 때문에 ::after 가상요소 선택자와 clear 속성을 활용하여 그 범위를 포함시킨다.

가상요소선택자를 통해 content 를 생성하면 가샹요소 선택자는 해당 요소안의 가상의 콘텐츠를 생성하게 된다.

figure 태그를 사용할 때 인라인 요소, 인라인 블록 요소를 감싸고 있을 땐, 높이가 자식 요소의 높이만큼 보장하지 않을 수 있다. 이때는 figure 태그에 font-size:0;을 하거나
자식 요소를 블록 요소로 변경하면 된다.

grid를 사용할 때 크기를 1fr 로 설정할 수 있는데 이 fr 은 남은 자유공간을 총 fr의 크기의 1fr 만큼 크기를 가진다는 것이다.
