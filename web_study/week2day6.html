5장 CSS 선택자 다루기

5.1 기본 선택자 사용하기

5.1.1 전체 선택자

전체 선택자는 HTML 에서 사용할 수 있는 모든 요소를 한 번에 선택자고 지정하는 방법으로, * 기호로 표시한다. 

                    
*{
    color:red;
}

5.1.2 태그 선택자

태그 선택자는 HTML 태그명으로 선택자를 지정하는 방법이다. 선택자에 지정된 태그명과 일치하는 모든 요소를 한 번에 선택한다. 

                    
p{
    color:red;
}

5.1.3 아이디 선택자

아이디 선택자는 HTML 태그에서 사용할 수 있는 id 속성값을 이용해 선택자를 지정하는 방법이다. 이때 속성값 앞에는 # 기호를 붙인다. 

                    
#title{
    color:red;
}

5.1.4 클래스 선택자

클래스 선택자는 HTML 태그에서 사용할 수 있는 class 속성값을 이용해 선택자를 지정하는 방법이다. 이때 속성값 앞에는 . 기호를 붙인다. 

                    
.red{
    color:red;
}
.blue{
    color:blue;
}

5.1.5 기본 속성 선택자

속성 선택자는 HTML 태그에서 사용할 수 있는 속성과 값을 사용해 선택자를 지정하는 방법이다. 

                    
a[href]{
    color:red;
}                   
                
                    
a[target="_blank"]{
    color:red;
}
이런 방식으로 속성과 속성값까지 지정할 수 있으며, 클래스 선택자나 아이디 선택자도 속성 선택자에 포함할 수 있다. 
                    
#title[href]{
    color:red; /*id 속성값이 title 이면서 href 속성이 있는 요소 선택*/
}

5.1.6 문자열 속성 선택자

문자열 속성 선택자는 태그가 가진 속성값이 특정한 문자열과 일치하는 요소를 선택자로 지정하는 방법이다.

5.2 조합 선택자 사용하기

5.2.1 그룹 선택자

그룹 선택자는 여러 선택자를 하나로 그룹 지을 때 사용한다. 선택자와 선택자는 , 기호로 구분한다. 

                    
p, #title, .red{
    color:red;
}

5.2.2 자식 선택자

자식 선택자는 부모 요소의 하위에 있는 자식 요소에 스타일을 적용할 때 사용한다. 2개 이상의 선택자가 사용되며, 선택자와 선택자는 > 기호로 구분한다. 

                    
.box > p{
    color:red; /* class 속성 값이 box인 요소와 자식 관계에 있는 p 태그를 선택*/
}

5.2.3 하위 선택자

하위 선택자는 선택자의 범위를 특정 부모 요소의 하위 요소로 한정하는 방법이다. 2개 이상의 선택자를 사용하고, 선택자와 선택자는 공백으로 구분한다. 

                    
#div p{
    color:red; /* div 태그 하위에 있는(자손인) p 태그에만 적용*/
}                   
                
                    
<div>
    <p>lorem1</p>
    <ul>
        <li>
            <p>lorem2</p> 
        </li>
        <li>
            <p>lorem3</p>
        </li>
    </ul>
</div> 
<p>lorem4</p> <!--div 태그의 하위에 있지 않아 적용 x-->

5.2.4 인접 형제 선택자

인접 형제 선택자는 앞에서 지정한 선택자 요소 바로 다음에 있는 형제 관계 요소를 선택자로 지정한다. 2개 이상의 선택자를 사용하고, 선택자와 선택자는 + 기호로 구분한다. 

                    
h1 + h2{
    color:red; /* h1 태그와 형제 관계이며 인접한 h2 태그에만 적용*/
}

5.2.5 일반 형제 선택자

일반 형제 선택자는 이전 선택자 뒤에 오는 형제 관계 요소를 모두 선택자로 지정한다. 2개 이상의 선택자를 사용하며, 선택자와 선택자는 ~ 기호로 구분한다. 

                    
h1 ~ h2{
    color:red; /* 이전 요소 다음에 오는 대상 요소가 형제 관계면 적용*/
}
인접 형제 선택자와 일반 형제 선택자 모두 이전 요소보다 먼저 등장하는 형제 관계의 요소는 선택자 지정 대상이 아니다.

5.3 가상 요소 선택자 사용하기

가상 요소 선택자는 HTML 문서에 명시적으로 작성된 구성 요소는 아니지만, 마치 존재하는 것처럼 취급해 선택하는 선택자 지정 방법을 말한다. 앞에 :: 기호를 붙여 사용하며, 기준 선택자와 함께 사용한다. 

                    
p::before{
    content: '<before>'
} 
p::after{
    content: '<after>' /*p 태그의 콘텐츠 앞과 뒤에 content 속성으로 새로운 텍스트를 추가*/
}

5.4 가상 클래스 사용하기

기준 요소:가상 클래스 선택자 {css 코드}

5.4.1 링크 가상 클래스 선택자

링크 가상 클래스 선택자는 a 태그에서 발생할 수 있는 링크 상태를 이용해 선택하는 방법이다. 

                    
a:link{
    color:orange;  /*한 번도 방문한 적이 없는 링크*/
} 
a:visited{
    color:green; /*한 번이라도 방문한 적이 있는 링크*/
}

5.4.2 동적 가상 클래스 선택자

동적 가상 클래스 선택자는 사용자의 어떤 행동에 따라 동적으로 변하는 상태를 이용해 선택자를 지정하는 방법이다. 

                    
p:hover{
    color:red;  /* 요소에 마우스를 올리면 해당 요소가 선택자로 지정됨*/
} 
button:active{
    color:green; /* 태그를 마우스로 클릭하면 해당 태그가 선택자로 지정됨*/
}

5.4.3 입력 요소 가상 클래스 선택자

입력 요소 가상 클래스 선택자는 입력 요소의 특정 상태를 이용해 선택자로 지정하는 방법이다. 

                    
input:focus{
    color:red;  /* 입력 요소에 커서가 활성화되면 스타일을 적용한다.*/
} 
input:check + label{
    color:red;  /* 체크박스가 표시되어 있고, 인접한 형제 요소인 label 태그에 스타일을 적용한다.*/
}
input:disabled, button:disabled{
    background-color:red;  /* 상호작용 요소가 비활성화 되어 있으면 스타일을 적용한다.*/
}
input:enabled, button:enabled{
    background-color:red;  /* 상호작용 요소가 활성화 되어 있으면 스타일을 적용한다.*/
}

5.4.4 구조적 가상 클래스 선택자

구조적 가상 클래스 선택자는 HTML 태그의 사용 위치, 다른 태그와의 관계에 따라 요소를 선택하는 방법이다. 

                    
li:first-child{
    color:red;  /* li 태그의 자식 요소 중에서 첫 번째 요소를 선택함.*/
} 
li:last-child{
    color:red;  /* li 태그의 자식 요소 중에서 마지막 요소를 선택함.*/
}                   
                
                    
p:nth-child(n){
    color:red;  /* p 요소가 부모의 모든 자식 요소 중 n번째 자식 요소가 맞으면 선택함.*/
} 
p:nth-last-child(n){
    color:red;  /* p 요소가 부모의 모든 자식 요소 중 마지막에서 n번째 자식 요소가 맞으면 선택함.*/
}                   
                
                    
p:nth-of-type(n){
    color:red;  /* 부모의 자식 요소 중 n번째로 등장하는 p 요소를 선택함.*/
} 
p:nth-last-of-type(n){
    color:red;  /* 부모의 자식 요소 중 마지막에서 n번째로 등장하는 p 요소를 선택함.*/
}                   
                
                    
p:first-of-type(n){
    color:red;  /* 부모의 자식 요소 중 첫 번째로 등장하는 p 요소를 선택함.*/
} 
p:last-of-type(n){
    color:red;  /* 부모의 자식 요소 중 마지막에 나오는 p 요소를 선택함.*/
}

5.5 다양한 선택자 조합하기

다양한 선택자를 서로 조합해서 사용할 수 있으며 가상 클래스와 하위 선택자, 자식 선택자를 조합한 형태도 가능하다. 예시는 다음과 같다. 

                    
div.box{
    backgrond-color:red;  /* class 속성값이 box인 div 태그 */
} 
section#main{
    backgrond-color:red;  /* id 속성값이 main인 section 태그 */
}
div:hover button{
    backgrond-color:red;  /* div 태그에 마우스를 올린 상태일때, 해당 태그 하위에 있는 button 태그를 선택 */
}

셀프체크 1

1. p 태그를 모두 선택합니다. 

                    
p{
}

2. id 속성값이 title인 태그를 선택합니다. 

                    
#title{
}

3. class 속성값이 box 이면서 id 속성값이 title 인 태그를 선택합니다. 

                    
.box#title{
}

4. section 태그의 하위에 있는 모든 div 태그를 선택합니다. 

                    
section div{
}

5. label 태그와 인접한 형제 관계인 input 태그를 선택합니다. 

                    
label + input{
}

6. 링크 대상 경로가 http://www.naver.com 인 a 태그의 하위에 있는 span 태그를 모두 선택합니다. 

                    
a[href="http://www.naver.com"] span{
}
