10장 자바스크립트 함수 다루기

10.1 함수란

함수는 어떤 목적을 가지고 작성한 코드를 모아 둔 블록문이다. 코드를 함수로 만들면 함수를 호출해 함수 내부에 모아 둔 여러 줄의 코드를 한 번에 실행할 수 있어 재사용하기가 편하다는 특징이 있다.

function 키워드와 식별자, 소괄호와 함께 묶으면 함수가 생성되며, 이를 함수를 정읳나다고 한다.

10.2 함수를 정의하는 방법

함수는 함수 선언문, 함수 표현식, 화살표 함수 등을 사용해 정의한다.

10.2.1 함수 선언문으로 함수 정의하기

함수 선언문은 function 키워드로 함수를 정의하는 방법이다. 

                    
function gugudan(){
    for(let i = 1; i <= 9; i++){
        console.log(`3 * ${i} = {3 * i}`);
    }
}
와 같은 방법으로 gugudan 을 식별자로 사용하여 함수를 만들 수 있다. 함수를 실행할때는 함수명과 소괄호를 열었다가 닫아주는 형식으로 사용하면 된다.

10.2.2 함수 표현식으로 함수 정의하기

함수는 객체에서 파생된 자료형이기 때문에 함수도 변수에 할당할 수 있다. 변수에 할당하는 함수에 식별자가 있으면 네이밍 함수, 없으면 익명함수라고 구분한다.

함수 선언문과는 다르게 function 키워드 다음에 오는 식별자로 함수를 호출하지 않고, 할당한 변수명으로 호출한다. 함수 식별자로 호출하려하면 참조 오류가 난다.
함수 표현식으로 함수를 정의할 때는 const 키워드를 사용하는 것이 좋다. 재정의해서도, 재할당해서도 안되는 경우가 많기 때문이다.

10.2.3 화살표 함수로 함수 정의하기

화살표를 사용해 함수를 정의할 수 있따. 화살표 함수는 익명함수로만 정의할 수 있어서 함수를 호출하려면 정의된 함수를 변수에 할당하는 방법을 사용해야 한다. 

                    
const gugudan = () => {
    for(let i = 1; i <= 9; i++){
        console.log(`3 * ${i} = {3 * i}`);
    }
}

10.3 함수 기능 확장하기

10.3.1 매개변수와 인수

함수에는 매개변수와 인수를 정의할 수 있다. 매개변수는 함수를 정의할 때 외부에서 전달하는 데이터를 함수에서 받을 수 있도록 정의하는 변수이고,
인수는 정의한 함수를 호출할 때 소괄호 안에 전달하고 싶은 데이터를 뜻한다. 

                    
function gugudan(dan){
    for(let i = 1; i <= 9; i++){
        console.log(`${dan} * ${i} = {dan * i}`);
    }
}  
gugudan(3);
gugudan(5);
이처럼 매개변수와 인수를 함수에 적용해서 사용할 수 있다.

10.3.2 매개변수의 특징

  • 명명 규칙

    매개변수는 일반적으로 변수와 같으며 매개변수의 식별자도 변수의 식별자 명명규칙에 따른다.

  • 데이터 전달

    매개변수는 함수 호출 시 전달되는 데이터와 일대일로 대응되어 데이터가 할당된다. 첫 번째 인수는 첫 번째 매개변수로, 두 번째 인수는 두 번째 매개변수로 할당된다.
    만약 인자를 적지 않고 호출할 시 undefined 값으로 할당되어 코드를 실행해도 오류가 발생하지 않는다.

  • 기본값 할당

    매개변수에 직접 데이터를 할당하는 방식으로 순수하게 기본값을 지정할 수 있다.

10.3.3 return 문

함수를 호출할 때 데이터를 전달하려면 매개변수를, 함수 외부로 데이터를 반환할 때는 return 문을 사용해야 한다.

return 문으로 함수 내부 데이터를 함수 외부로 전달하게 되면 함수를 호출한 곳으로 데이터가 전달되며 이를 반환한다라고 하고, 이 데이터를 반환값이라 한다.
표현식으로도 데이터를 반환할 수 있으며, return 문에 데이터를 반환하지 않으면 단순히 함수 실행을 종료하는 역할을 하게 된다. 값은 undefined ㄱ밧을 반환한다.
