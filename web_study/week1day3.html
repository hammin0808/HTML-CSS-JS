실무에서 자주 사용하는 HTML 필수 태그 다루기

3.1 텍스트 작성하기

3.1.1 hn 태그

hn 태그는 제목이나 주제를 나타내는 텍스트를 표현할 떄 사용한다. h1 부터 h6 까지 있으며 1에서 6으로 숫자가 커질수록 중요도와 글씨 크기가 떨어진다.
또한, hn 태그로 작성된 텍스트는 검색 엔진에서 키워드로 인식하며 단계적으로 사용해야 한다.

3.1.2 p 태그

p 테그는 본문의 문단을 작성할 때 사용한다.

3.1.3 br 태그

br 태그는 문단에서 줄 바꿈할 떄 사용한다. HTML은 모든 명령이나 지시를 태그로 해야 하기 때문에 코드내에서 줄 바꿈을 한다고 해도, br 태그를 명시적으로 사용하지 않았으면 줄 바꿈이 되지 않는다.

3.1.4 blockquote 태그

blockquote 태그는 출처에서 인용한 문단 단위의 텍스트를 작성할 때 사용한다. 또한 반드시 한 개 이상의 p 태그를 포함해야 한다. 형식은 다음과 같다.
형식 : <blockquote cite="출처 URL">문단단위 인용문</blockquote>

3.1.5 q 태그

q 태그는 문단 안에 텍스트 단위의 짧은 인용문을 작성할 때 사용한다. q 태그를 사용한 콘텐츠는 큰따옴표로 묶이게 된다. 형식은 다음과 같다.
형식 : <q cite="출처 URL">짧은 인용문</q>

3.1.6 ins와 del 태그

ins 태그는 새로 추가된 텍스트임을 나타낼 떄 사용하고, del 태그는 기존에 있던 텍스트가 삭제된 텍스트임을 나타낼 때 사용한다. ins 태그에는 밑줄이, del 태그에는 취소선이 생긴다.

3.1.7 sub와 sup 태그

sub 태그와 sup 태그는 각각 아래 첨자, 위 첨자에 해당하는 텍스트를 작성할 때 사용한다.

3.2 그룹 짓기

3.2.1 div 태그

div 태그는 블록 요소와 인라인 요소를 그룹으로 묶을 때 사용한다. 관련 있는 요소를 영역별로 그룹을 지어 작성하여 구조를 깔끔하게 정리해준다.

3.2.2 span 태그

span 태그는 인라인 요소를 그룹으로 묶을 때 사용한다. 텍스트도 인라인 요소이기 때문에 태그 안의 내부 콘텐츠를 span 태그로 그룹을 지어 공간을 분할할 수 있고, 이에 CSS 스타일을 적용할 수 있게 된다.

3.3 목록 만들기

3.3.1 ul 태그

ul 태그는 순서가 없는 비순서형 목록을 생성할 때 사용한다. 목록 내용은 li 태그로 구성한다. ul 태그를 사용하면 목록 내용마다 글머리 기호가 붙는다.

판매 중인 과일

  • 바나나
  • 수박
  • 사과

3.3.2 ol 태그

ol 태그는 순서형 목록을 생성할 때 사용한다. 목록 내용은 li 태그로 구성한다. ol 태그를 사용하면 목록 내용마다 번호가 붙는다.

판매 중인 인기 과일

  1. 바나나
  2. 수박
  3. 사과

3.3.3 dl 태그

dl 태그는 정의형 목록을 만들 때 사용한다. 정의형 목록은 용어와 용어 설명을 나열한 형태의 목록이다. 목록 내용은 dt 태그로 용어를, dd 태그로 용어 설명을 작성한다.

HTML
HTML은 Hyper Text Markup Language의 약자로 웹 문서의 구조를 설계하기 위한 목적으로 개발된 언어입니다.

3.4 링크와 이미지 넣기

3.4.1 a 태그

a 태그는 HTML에서 내부나 외부 링크를 생성한다. 대상 경로를 의미하는 href 속성을 필수로 사용해야 하고, 그외 target, title 속성을 선택해서 사용할 수 있다. 형식은 다음과 같다.
형식 : <a href="대상경로" target="링크 연결 방식" title="링크 설명"></a>

  • href 속성

    href 속성은 a 태그로 생성하는 링크의 대상 경로를 입력할 때 사용한다. 속성값은 대상 경로의 주소이거나 내부 문서의 id 속성값일 수 있다. 대상 경로가 명확하지 않을 때는 속성값으로 #을 넣는다.

  • target 속성

    target 속성은 a 태그로 링크를 생성할 때 대상이 연결되는 방식을 지정한다. 속성값으로 _blank, _parent, _self, _top을 사용할 수 있다.

  • title 속성

    title 속성에는 링크를 설명할 수 있는 텍스트를 작성한다. a 태그의 콘텐츠만으로 포현하지 못한 설명을 적는다.

3.4.2 img 태그

HTML에서 이미지 객체를 삽입하고 싶을 때 사용하며, img 태그는 src 속성과 alt 속성으로 구성된다. 형식은 다음과 같다. 형식 : <img src="이미지 경로" alt="이미지 설명">

  • src 속성

    src는 삽입하려는 이미지의 경로를 입력하는 속성이다. HTML에서 이미지 경로는 항상 웹 브라우저에서 실행되는 HTML 파일의 위치가 기준이다. 형식은 다음과 같다.
    형식 : <img src="./images/beach.jpg">
    이미지 경로에 사용되는 기호는 다음과 같다.
    ./ : 현재 폴더 ../ : 상위 폴더

  • alt 속성

    alt 속성에는 삽입한 이미지 객체를 설명할 수 있는 텍스트를 넣는다. 형식은 다음과 같다.
    형식 : <img src="./images/beach.jpg" alt="아름다운 해변">

3.4.3 이미지 링크

a 태그 안에는 텍스트뿐만 아니라 여러 요소가 올 수 있다. 그중 img 태그를 a 태그 안에 사용하면 이미지를 클릭 했을때 특정 링크로 이동하게 할 수 있다.

3.5 텍스트 강조하기

3.5.1 strong 태그

strong 태그는 텍스트의 의미를 강조하고 싶을 때 사용하며 텍스트가 굵게 표시된다. 중첩해서 사용할 수 있으며 효과는 동일하지만 구조적으로 더 중요한 부분임을 코드에 명시할 수 있다.

em 태그

em 태그는 텍스트의 의미를 강조하고 싶을 때 사용하며 텍스트가 기울어져 보이게 된다.
