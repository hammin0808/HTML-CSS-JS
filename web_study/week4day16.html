11장 자바스크립트 객체 다루기

11.3 표준 내장 객체 사용하기

표준 내장 객체란 자바스크립트에 기본으로 내장되어 있어서 스코프의 위치를 따지지 않고 모든 영역에서 공통으로 사용할 수 있는 객체를 말한다.

11.3.1 문자열을 다루는 String 객체

String 객체는 기본 자료형에서 문자열을 다룬다. 문자열에서 사용할 수 있는 속성과 메서드가 정의되어 있다.
length 속성을 사용하면 문자열의 길이를 알 수 있다.
includes 메서드를 사용하면 특정 문자열이 포함되어 있는지 확인할 수 있다. 이는 indexOf 메서드를 사용해도 확인할 수 있는데, 만약 찾지 못하면 -1 을 반환한다.

11.3.2 배열을 다루는 Array 객체

Array 객체는 기본 자료형 중 배열을 다루는 객체로 배열에서 사용할 수 있는 많은 속성과 메서드가 정의되어 있다.
Array 객체의 메서드를 보면 파괴적 메서드와 비파괴적 메서드로 구분이 된다. 파괴적 메서드는 메서드를 사용했을 때 원본 데이터를 변경하는 메서드를 의미하고, 비파괴적 메서드는 원본을 변경하지 않는 메서드를 의미한다.

11.3.3 날짜와 시간을 다루는 Date 객체

Date 객체에는 날짜 및 시간과 관련있는 메서드가 정의되어 있다.

  • 인스턴스 만들기

    Date 객체를 사용하려면 먼저 Date 객체를 호출해 날짜와 시간 데이터를 생성해야 한다. new 키워드를 사용해 Date 객체를 호출하면 현재 날짜와 시간 정보를 기반으로 날짜와 시간 데이터가 생성된다.
    이 데이터를 인스턴스라고 하며 date 변수에 할다오딘 데이터는 Date 객체의 인스턴스이다.

  • 메서드로 날짜와 시간 정보 가져와서 설정하기

    해당 객체로 인스턴스를 만들었을 때 현실에서 사용하는 형식과 다르게 생성되기 때문에 메서드로 일반적인 형식으로 가공해야 한다. get으로 시작하는 메서드로 날짜와 시간 정보를 가져오고, set으로 시작하는 메서드로 날짜와 시간 정보를 설정한다.

  • 날짜 간격 계산하기

    getTime 메서드는 1970년 1월1일 12:00 이후의 시간을 밀리초 단위로 반환한다. 따라서 두 날짜 사이의 간격을 해당 반환결과를 사용해서 구할 수 있는데, 구한 밀리초를 24 x 60 x 60 x 1000 로 나누면 두 날짜 사이의 간격을 일수로 환산할 수 있다.

11.3.4 수학 연산을 다루는 Math 객체

Math 객체에는 여러 가지 수학 연산을 수행하는 메서드가 정의되어 있다. Date 객체와 달리 인스턴스를 생성하지 않고, Math 객체에 바로 메서드를 사용하면 된다.

  • 난수 구하기

    Math 객체의 random() 메서드는 0 이상 1 미만의 난수를 반환한다. 같은 코드라도 실행할 때마다 값이 달라지며, 구하고 싶은 난수의 범위가 있다면 해당 값을 곱하면 된다.

11.4 브라우저 객체 모델 사용하기

브라우저 객체 모델이란 웹 브라우저에서 제공하는 객체로 자바스크립트 언어 사양에 포함되어 있지 않는다. 최상위 객체로 window 가 있으며, 그 아래로 document, location, history, navigator, screen 객체가 있다.

11.4.1 window 객체의 속성과 메서드

window 객체에는 웹 브라우저의 기능과 요소들을 제어할 수 있는 여러 속성과 메서드가 있다.

11.4.2 windotw 객체의 기본 속성 사용하기

window 객체의 속성은 주로 웹 브라우저의 창과 관련 있는 속성이 많다.

11.4.3 웹 브라우저에서 새 창 제어하기

window 객체의 open() 메서드는 웹 브라우저에서 새로운 창을 여는 데 사용한다. 매개변수로 창의 경로, 이름, 속성을 전달받으며 특정 행동으로 원하는 파일이 새 창에서 열리게 할 수 있다.

11.4.4 웹 브라우저의 스크롤 이동하기

window 객체의 scrollTo() 메서드는 웹 브라우저의 스크롤 위치를 특정 좌표로 이동하게 한다. scrollBy() 메서드는 웹 브라우저의 스크롤을 현재 위치에서 상대적인 위치로 이동하게 한다.
