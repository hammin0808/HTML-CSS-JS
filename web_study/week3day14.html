10.4 함수의 특징 이해하기

10.4.1 스코프

기본적으로 함수 내부의 변수를 함수 외부에서 참조할 수 없다. 스코프는 변수나 함수와 같은 참조 대상 식별자를 찾아내기 위한 규칙으로, 그 범위가 다르기 때문에 참조하지 못한 것이다.

  • 함수 스코프

    함수 스코프는 함수에서 정의한 블록문만 스코프의 유효 범위로 인정하는 방식이다. 함수 내부는 지역스코프, 함수 외부는 전역 스코프 영역이 된다.
    전역 스코프에 해당하는 변수를 참조하면 내부나 외부와 상관없이 오류가 발생하지 않으나, 지역 스코프에 해당하는 변수를 외부에서 사용하려 하면 오류가 발생한다.

  • 블록 스코프

    블록 스코프는 {} 로 구성된 블록문 기준으로 스코프의 유효 범위를 나누는 방식이다. 이 방식은 let 과 const 키워드로 선언한 변수에 한해서만 적용된다.

  • 참조 우선순위

    let, const 키워드는 같은 식별자의 중복 선언이 불가능하지만, 정확하게는 같은 스코프 영역에서 중복 선언이 불가능한 것이다.

10.4.2 함수 호이스팅

호이스팅은 코드를 선언과 할당으로 분리해 선언부를 자신의 스코프 최상위로 끌어올리는 것을 말한다. var 키워드로 선언한 변수에만 적용되며 함수에도 적용된다.

함수 표현식이나, 화살표 함수 방식으로 정의하면 변수에 할당하는 것이기 때문에 함수 선언부를 변수를 사용한 후에 작성하면 변수가 함수가 아니라는 오류 메시지가 나오게 된다.

10.5 즉시 실행 함수 사용하기

즉시 실행 함수는 함수를 정의하면서 동시에 실행까지 하는 함수이다. 일반적으로 함수를 선언하면 전역 스코프에 정의된다. 그리고 프로그램이 종룓되기까지 메모리에서 사라지지 않는다.
만약 해당 함수를 한 번만 사용할 함수인데, const 키워드 때문에 함수에 사용한 식별자를 더 이상 사용할 수 없게 되면 이를 전역 스코프가 오염됐다고 표현한다.
이런 경우에 즉시 실행 함수로 함수를 정의하면 전역스코프가 오염되는 것을 방지할 수 있다.

셀프체크1

                    
function area(r){
    return r * r * 3.14;
}

셀프체크2

                    

function f2(arr){
    let temp = 0;
    for(let i = 0; i <= arr.length; i++){
        if(temp <= arr[i]) temp = arr[i];
    }
    return temp;
}

셀프체크3

                    
function f3(cm, kg){
    let bmi = kg / (cm / 100)**2;
    if (bmi >= 26){
        return "비만";
    }
    else if((24 <= bmi) && (bmi <= 25)){
        return "과체중";
    }
    else if((18.5 <= bmi) && (bmi <= 23)){
        return "정상";
    }
    else return "저체중";
}
